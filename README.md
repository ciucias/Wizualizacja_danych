Projekt z przedmiotu metody wizulizacji i pozyskiwania danych.
Proste wizualizacje poazujące jak wyglądają dane w zbiorze dotyczącym aut osobowych.
W projekcie została wykorzystana głównie biblioteka tidyverse do stworzenia podstawowych wykresów pokazującuych jak zachowują się dane w tym zbiorze oraz bardziej zaawansowane reagujące na użytkownika.
PROJEKT - https://ciucias.github.io/Wizualizacja_danych/
